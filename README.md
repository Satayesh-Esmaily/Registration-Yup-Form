# Registration/Sign in 
 Register & Sign In (React + Tailwind + React Hook Form + Yup)

This project contains Register and Sign In forms built with React, using React Hook Form and Yup for form handling and validation.

All forms are component-based and styled with Tailwind CSS

Features

    Separate and reusable Register and Sign In forms
    Realtime validation with React Hook Form + Yup
    Inline error messages under each input field
    Buttons are disabled until the form is valid
    
Reusable components:

    FormInput, FormCheckbox, FormButton, FormSuccess
    AuthLayout, AuthFooter, AuthHeader
    Footer links for switching between forms
    Clean and responsive design with Tailwind CSS\

Installation & Running

    1. Navigate to the Folder 
     cd Registration
    2. Install dependencies
     npm install (i)
    3. Run the Project
     npm run dev 
    and open the localhost on your browser
